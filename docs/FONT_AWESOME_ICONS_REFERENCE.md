# Font Awesome Icons Reference

This project uses Font Awesome icons. You can use any of these icon classes in your templates.

## How to Use Icons

Just add the icon class to an `<i>` tag:

```html
<i class="fa fa-rocket"></i> Launch
```

Or in dropdown menus:

```html
<li>
  <a href="/your-url/">
    <i class="fa fa-rocket"></i>
    Your Menu Item
  </a>
</li>
```

## Icon Variants

### Solid (Default)
```html
<i class="fa fa-home"></i>
<i class="fa-solid fa-home"></i>  <!-- Same as above -->
```

### Regular (Outlined)
```html
<i class="fa-regular fa-heart"></i>
```

### Brands (Logo Icons)
```html
<i class="fa-brands fa-github"></i>
<i class="fa-brands fa-stripe"></i>
<i class="fa-brands fa-google"></i>
```

## Icon Sizes

Add size classes:

```html
<i class="fa fa-home fa-xs"></i>     <!-- Extra small -->
<i class="fa fa-home fa-sm"></i>     <!-- Small -->
<i class="fa fa-home"></i>            <!-- Default -->
<i class="fa fa-home fa-lg"></i>     <!-- Large -->
<i class="fa fa-home fa-2x"></i>     <!-- 2x size -->
<i class="fa fa-home fa-3x"></i>     <!-- 3x size -->
```

Or use Tailwind classes:

```html
<i class="fa fa-home h-4 w-4"></i>   <!-- 16px -->
<i class="fa fa-home h-5 w-5"></i>   <!-- 20px -->
<i class="fa fa-home h-6 w-6"></i>   <!-- 24px -->
```

## Spinning Icons (for Loading States)

## Finding More Icons

Visit: **https://fontawesome.com/search?o=r&m=free**

1. Search for what you need (e.g., "email", "truck", "calendar")
2. Click on the icon you like
3. Copy the class (e.g., `fa-solid fa-envelope`)
4. Use it in your template!

## Common Patterns in Your App

**For menu items:**
```html
<i class="fa fa-icon-name h-4 w-4"></i>
```

**For buttons:**
```html
<i class="fa fa-icon-name mr-2"></i>
```

**For status badges:**
```html
<i class="fa fa-check mr-1"></i>
```
